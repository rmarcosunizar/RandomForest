Random Forest

Introduccin a Random Forest

Random Forest es un algoritmo de prediccin que utiliza un conjunto de normas binarias para calcular un valor objetivo. Este algoritmo puede ser utilizado para clasificacin (variables categrica) o regresin (variables continuas).

El algoritmo Random Forest esta basado en el mtodo de rboles de decisin (CART - Classification and Regression Trees).

El algoritmo generador de rboles:

  • Determina que variable utilizar para dividir un nodo y el valor a utilizar.
  • Decide cuando parar (nodo terminal) o continuar la divisin.
  • Utiliza diferentes aproximaciones para determinar el mejor umbral de separacin en un nodo.
  • Asigna una clase a cada terminacin.

Random Forest es un clasificador de ensamblado (ensemble classifier). Este tipo de clasificadores se basan en calibrar varios submodelos, en este caso rboles de decisin, y posteriormente agregar los resultados de los distintos modelos para determinar la prediccin final.

Adems, RF provee informacin sobre la precisin del modelo y la importancia de las variables predictivas.

Consideraciones especficas de funcionamiento

  1. Para cada uno de los rboles ajustados en el modelo, e algoritmo selecciona aleatoriamente 2/3 del conjunto total de datos (poblacin) como muestra de calibracin en sucesivos pasos (cada paso representa un rbol de decisin) para ajustar el modelo.

  2. La muestra restante (denominada Out-Of-Bag o OOB). Esta muestra es usada por el propio algoritmo para medir el error y la importancia de las variables.

  3. La asignacin del valor final (prediccin) se hace en funcin de los votos obtenidos de los diferentes rboles.

Cada rbol es construido usando el siguiente proceso:

  • SeaNel numero de casos de prueba,Mes el numero de variables en el modelo.
  • Seamel numero de variables de entrada a ser usado para determinar la decisin en un nodo dado(\(m<M\)).
  • Elegir un conjunto de entrenamiento para este rbol y usar el resto de los casos (OOB) de prueba para estimar el error.
  • Para cada nodo del rbol, elegir aleatoriamentemvariables en las cuales basar la decisin. Calcular la mejor particin a partir de lasmvariables del conjunto de entrenamiento.

Para la prediccin un nuevo caso es empujado hacia abajo por el rbol. Luego se le asigna la etiqueta del nodo terminal donde termina. Este proceso es iterado por todos los arboles en el ensamblado, y la etiqueta que obtenga la mayor cantidad de incidencias es reportada como la prediccin.

Random Forest en R

El paquete randomForest provee las funciones para trabajar con modelos Ranfom Forest en R.

Principalmente nos centraremos en usar la funcin randomForest(), que es la que permite al ajuste de modelos de regresin o clasificacin.

#install.packages('randomForest')
library(randomForest)
help(randomForest)

La sintaxis del argumento formula es la misma que la vista en otros modelos de regresin.

No obstante existen otros argumentos especficos como:

  • mtry: nmero de variables consideradas para separar cada nodo.
  • importance: determina si se calculo o no la importancia de las variables predictivas en el modelo.
  • ntree: nmero de rboles utilizados para generar la prediccin.
